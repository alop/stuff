Description
===========

A script to ensure nova-compute instances are actually running in libvirt.

Requirements
============

Platform
--------

* Debian, Ubuntu
* RHEL, CentOS, Fedora

Tested on:

* Ubuntu 11.04

Attributes
==========

* `virsher['script_dir']` - Where do scripts get installed. Defaults to '/usr/local/bin'
